このページでは、アプリの使い方、よくある質問、お問い合わせ方法などをご案内しています。 お困りの際は、まずよくある質問をご確認ください。

📱 アプリについて

ブクレジは、お気に入りの作家の新刊発売日を自動で通知し、 カレンダーで管理できるiOSアプリです。

📖 発売予定リスト：登録作家の新刊を月別に表示
📅 カレンダー表示：発売予定を視覚的に管理
👤 お気に入り作家：作家ごとの通知ON/OFF設定
🔔 プッシュ通知：新刊発表・発売日リマインダー

❓ よくある質問（FAQ）

Q1. 通知が届きません

確認事項：

  • アプリの通知設定がONになっているか確認してください（設定画面）
  • iOSの設定 → 通知 → ブクレジ で通知が許可されているか確認
  • 作家ごとの通知設定がONになっているか確認（お気に入り画面）
  • インターネット接続を確認してください
Q2. 作家を追加できません

対処方法：

  • 作家名を正確に入力してください（フルネーム推奨）
  • インターネット接続を確認してください
  • アプリを再起動してみてください
  • 作家名の表記を変えて検索してみてください（例：「東野圭吾」→「ひがしのけいご」）
Q3. 新刊情報が更新されません

対処方法：

  • リリースリスト画面で下に引っ張ってリフレッシュ
  • 手動更新は5分に1回の制限があります
  • 自動更新は毎日実行されます
  • アプリを再起動してみてください
Q4. アカウントを削除したい

削除手順：

  1. アプリを開く
  2. 設定画面を開く
  3. 「アカウント削除」をタップ
  4. 確認画面で「削除」を選択

注意：削除後、すべてのデータが完全に削除されます。この操作は取り消せません。

Q5. 購入リンクが開けません

対処方法：

  • インターネット接続を確認してください
  • Safariまたはブラウザアプリが正常に動作するか確認
  • アプリを最新バージョンに更新してください
Q6. パスワードを忘れました

リセット手順：

  1. ログイン画面で「パスワードを忘れた場合」をタップ
  2. 登録メールアドレスを入力
  3. 送信されたメールのリンクからパスワードをリセット

📖 使い方ガイド

初回セットアップ

  1. アプリをダウンロードしてインストール
  2. メールアドレスとパスワードでアカウント作成
  3. メール認証を完了
  4. お気に入り作家を追加
  5. 通知設定を確認

作家の追加方法

  1. 「お気に入り」タブを開く
  2. 画面右上の「+」ボタンをタップ
  3. 作家名を入力して検索
  4. 検索結果から作家を選択
  5. 通知設定を選択して「追加」

カレンダーへの追加

  1. 書籍詳細画面を開く
  2. 「カレンダーに追加」ボタンをタップ
  3. または、カレンダー画面で「一括追加」を使用
  4. iOSカレンダーアプリで確認

🐛 バグ報告

アプリに問題を発見された場合は、以下の方法でご報告ください：

  • GitHub Issues: https://github.com/kimihiko/book-release-notifier-docs/issues

バグ報告には、以下の情報を含めていただけると対応がスムーズです：

  • 再現手順
